Function_04 : Create a chat

RG01 : User should be authenticated to create a chat
RG02 : Only two persons can share on one chat
RG03 : A chat is related to one annonce
RG04 : If an annonce is deleted, then all the chat related to this annonce will be deleted

Function Create a chat is only available for authenticated user.
In the annonce detail activity, buyer can choose to send a message (if the seller has chose this option)
